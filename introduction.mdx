# Welcome to UI/UX Pro Max

UI/UX Pro Max is an AI skill that provides design intelligence for building professional UI/UX across multiple platforms and frameworks. It combines a powerful CLI tool with an intelligent design system generator to help you create beautiful, accessible interfaces.

## What's New in v2.0

### Intelligent Design System Generation

The flagship feature of v2.0 is the **Design System Generator** — an AI-powered reasoning engine that analyzes your project requirements and generates a complete, tailored design system in seconds.

The system performs 5 parallel searches across:
- **Product type matching** (100 categories)
- **Style recommendations** (67 styles)
- **Color palette selection** (96 palettes)
- **Landing page patterns** (24 patterns)
- **Typography pairing** (57 font combinations)

The reasoning engine then applies industry-specific rules to deliver a complete design system with pattern, style, colors, typography, effects, and anti-patterns to avoid.

## How It Works

<Steps>
  <Step title="You ask">
    Request any UI/UX task: build, design, create, implement, review, fix, improve
  </Step>
  <Step title="Design System Generated">
    The AI automatically generates a complete design system using the reasoning engine
  </Step>
  <Step title="Smart recommendations">
    Based on your product type and requirements, it finds the best matching styles, colors, and typography
  </Step>
  <Step title="Code generation">
    Implements the UI with proper colors, fonts, spacing, and best practices
  </Step>
  <Step title="Pre-delivery checks">
    Validates against common UI/UX anti-patterns
  </Step>
</Steps>
